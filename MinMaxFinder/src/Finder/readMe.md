Java ile klavyeden girilen N tane sayma sayısından en büyük ve en küçük sayıları bulan ve bu sayıları ekrana yazan program

**Sonuç**

```
Kaç tane sayı gireceksiniz?: 6
1. sayıyı girin: 23
2. sayıyı girin: 4556
3. sayıyı girin: 43
4. sayıyı girin: 2
5. sayıyı girin: 3
6. sayıyı girin: 4
En büyük sayı: 4556
En küçük sayı: 2

````

```
Kaç tane sayı gireceksiniz?: 5
```
